# 🚀 12-Week Roadmap

This roadmap is designed to **systematically master DSA, System Design, and real-world projects**.

---

## Phase 1: Fundamentals (Weeks 1–4)
**DSA Focus:** Arrays, Strings, Recursion, Sliding Window, Two Pointers  
**System Design Focus:** CAP Theorem, Load Balancing, Caching, Database Indexing

- Week 1: Arrays + CAP Theorem  
- Week 2: Strings + Load Balancing  
- Week 3: Sliding Window + Caching  
- Week 4: Recursion + Database Indexing

---

## Phase 2: Intermediate (Weeks 5–8)
**DSA Focus:** Linked List, Stack & Queue, Trees, Heaps, Binary Search  
**System Design Focus:** REST vs Messaging, Rate Limiter, URL Shortener, Notification System, Payment System

- Week 5: Linked List + REST vs Messaging  
- Week 6: Stack & Queue + Rate Limiter  
- Week 7: Trees + URL Shortener  
- Week 8: Heaps + Notification System

---

## Phase 3: Advanced & Projects (Weeks 9–12)
**DSA Focus:** Graphs, Dynamic Programming, Mixed Practice  
**System Design Focus:** Chat System, Scalable Backend, Real-world Projects

- Week 9: Binary Search + Payment System  
- Week 10: Graphs + Chat System  
- Week 11: Dynamic Programming + System Scaling  
- Week 12: Mixed DSA + System Design Review + Projects

---

## Notes
- Revise each DSA pattern **weekly**  
- Focus on **tradeoffs and scalability** in system design  
- Build **small projects** alongside learning  
- Track progress in `PROGRESS_TRACKER.md`
